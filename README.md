# Projeto destinado para a vaga de Front-end Developer na Zitrus

### Técnologias Utilizadas:
- JS
- Vue
- Node
- Express

## Casos de uso:
“Quero usar a API dos correios, para meus queridos clientes me informar o CEP e eu
preencher os restantes das informações necessárias.”

“A parada é simples, quero uma a aplicação SPA, com feedbacks visuais para sinalizar
operações no sistema, com validação de inserção de dados nos formulários. E não para por
aí, viu? Hoje com a LGPD, quero que o sistema tenha login e senha, tranquilo?”

de como executar o projeto está dentro do diretório "Back"
