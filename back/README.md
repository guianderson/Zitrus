# Projeto Zitrus (Back-end)

## Para rodar esse projeto:

1. Instale as dependências com o comando:
```
    npm i
```

2. Rode as migrações do banco de dados com o comando:
```
    npm run typeorm migration:run
```

3. Será gerado um arquivo com o nome "database.bd", este é o banco de dados SQLite do projeto.

4. Carregue o arquivo "database.bd" em um gerenciador de banco de dados. Como no exemplo abaixo: 

<img src="https://user-images.githubusercontent.com/47038612/152840451-0972bcf6-4449-4863-a462-81562e2af22f.gif"></img>

5. Inicie a aplicação com o comando:
```
    npm run start
```
